Introduction

Get started with URL Shortener, A URL shortener that builded with powerful tools to help you know who used your link, Even there's an API that you can use in your project.

Get started

Looking to quickly add url shortener to your project? Use our API, provided by free to help you make your project faster!

How to use

To use our API, do the following steps :

Make a Shortened URL

To make a Shortened URL you just need to put the code below inside your js file:


let ajxRqstData = 'url=' + 'Your URL' + '&password='+ 'Your password (optinal)';
let xhttp;

if (window.XMLHttpRequest){
    xhttp = new XMLHttpRequest();
}else{
    xhttp = new ActiveXObject('Microsoft.XMLHTTP');
}

xhttp.open('POST', 'make/', true);
xhttp.setRequestHeader("Content-type", "application/x-www-form-urlencoded");
xhttp.send(ajxRqstData);

xhttp.addEventListener('readystatechange', function () {
    if (xhttp.readyState == 4 && xhttp.status == 200){
        console.log(xhttp.responseText);
    }
});

And if you are using jQuery, you can simply add this to your js file:


$.post("make/", {
  url: 'Your URL', // add the url
  password: 'Your password'// add password if there is or else do not add it in the request
},
function(data){
  console.log(data);
});

Get logs of a shortened URL

To get logs of a shortened URL, just add this following block of code to your js file:


let ajxRqstData = 'url=' + 'Your URL' + '&password='+ 'Your password';
let xhttp;

if (window.XMLHttpRequest){
    xhttp = new XMLHttpRequest();
}else{
    xhttp = new ActiveXObject('Microsoft.XMLHTTP');
}

xhttp.open('POST', 'log/', true);
xhttp.setRequestHeader("Content-type", "application/x-www-form-urlencoded");
xhttp.send(ajxRqstData);

xhttp.addEventListener('readystatechange', function () {
    if (xhttp.readyState == 4 && xhttp.status == 200){
        console.log(xhttp.responseText); // This will return an Object that it has a Array
    }
});

And for jQuery, just do the below:


$.post("logs/", {
  "url" : 'Your URL',
  "password" : 'Your password'
  },
  function(data){
      console.log(data);
  }
);

Read fetched data

Read fetched data is simple, just like the following.

First, it's better to check the loaded data to make sure that data loaded properly:

Tip : In jQuery you can write the code below in done() method to make sure that it runs after that request is done.


if (typeof data == 'object'){
    readData();
}else{
    console.log('something went wrong!');
}

And here is the readData() function:

                    
function readData() {
  for (let x of data[0]){
      console.log(`
      Browser : ${x.browser},
      Date : ${x.date},
      Platform : ${x.platform},
      Referrer : ${x.referrer},
      Time : ${x.time}
      `);
  }
}

Here your work is done, Just remember that for get number of visitors you need to use `data[0].length`.
